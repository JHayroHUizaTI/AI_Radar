# Final Demo

The final demo should tell the truth about the current system.

Include:

- what AI Radar does;
- what data it uses;
- what is backed by evidence;
- what was tested;
- what remains pending;
- the preview URL when available;
- the PR or branch used for review.

Do not commit:

- rendered videos;
- raw recordings;
- large screenshots;
- generated presentation exports;
- temporary HyperFrames or Remotion build output.

## Demo Story

1. Show the signal contract.
2. Show the skill and parser tool.
3. Show source lanes and Notion source policy.
4. Show Supabase persistence contract.
5. Show the frontend reading fixture-backed signals.
6. Show tests and deploy smoke criteria.

## Honest Close

End with what is proven locally, what depends on connected services, and what should become the next issue.
