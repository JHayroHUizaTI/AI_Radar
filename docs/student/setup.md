# Setup de AI Radar

## Requisitos

- Node.js 22 o superior.
- pnpm 11 o superior.

## Comandos

```bash
pnpm install
pnpm lint
pnpm build
pnpm dev
```

La app corre en:

```text
http://localhost:4173
```

## Variables

Copia `.env.example` solo si necesitas probar integraciones live. No uses credenciales reales al inicio del curso.
